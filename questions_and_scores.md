# NBTI v8 终审版

> **NBTI 职场牛马人格测试** — 30题速测你的职场生存人格
>
> 三个维度，八种命运：
> - **N（牛马值）**：N+1 = 卷 / N-1 = 躺
> - **B（背锅值）**：B+1 = 扛 / B-1 = 闪
> - **T（套路值）**：T+1 = 深 / T-1 = 浅
>
> 每道题选一个最接近你真实反应的选项。别想太多，第一反应最准。

---

### Q1. 领导在团建上说"我简单讲两句"，一小时过去了还没停。你？

A. 认真听还记了笔记，万一以后考呢。 `(N:+1, B:+1)`
B. 频频点头，但脑子里在盘今晚吃什么。 `(N:+1, T:-1)`
C. 站起来提醒："领导，后面还有环节，要不先进行下一个？" `(N:-1, T:+1)`

## 计分规则

### 维度计算

每道题的选项影响恰好 **2个维度**（+1或-1），另一个维度不受影响。

30题答完后，分别汇总三个维度的总分：

| 维度 | 总分 > 0 | 总分 < 0 | 总分 = 0 |
|------|----------|----------|----------|
| **N（牛马值）** | 卷 | 躺 | 随机归入 |
| **B（背锅值）** | 扛 | 闪 | 随机归入 |
| **T（套路值）** | 深 | 浅 | 随机归入 |

### 八种人格

| 编码 | 人格名称 | 组合含义 |
|------|----------|----------|
| N+B+T+ | **赛博牛马** | 卷 + 扛 + 深 |
| N+B+T- | **卷心菜** | 卷 + 扛 + 浅 |
| N+B-T+ | **职场貔貅** | 卷 + 闪 + 深 |
| N+B-T- | **末日狂花** | 卷 + 闪 + 浅 |
| N-B+T+ | **人间清醒** | 躺 + 扛 + 深 |
| N-B+T- | **天选工具人** | 躺 + 扛 + 浅 |
| N-B-T+ | **不粘锅** | 躺 + 闪 + 深 |
